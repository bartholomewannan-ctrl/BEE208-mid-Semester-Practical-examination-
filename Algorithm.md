ALGORITHM
1.Start the program. 
2.Display the system title. 
3.Enter the event name and event date. 
4.Enter the number of participants. 
5.Set total expected amount, total paid amount, paid count, and unpaid count to zero. 
6.Repeat for each participant: 
*Enter participant name. 
*Enter participant ID or phone number. 
*Enter category: Student, Regular, VIP, or Guest. 
*Validate the category. If invalid, request it again. 
*Assign the registration fee: 
Student = GHS 20 
Regular = GHS 50 
VIP = GHS 100 
Guest = GHS 0 
*Enter payment status: Paid or Not Paid. 
*Validate the payment status. 
*Add the registration fee to total expected amount. 
*If payment status is Paid, add fee to total paid amount and increase paid count. 
*Otherwise, increase unpaid count. 
7.Display the final event registration report. 
8.Save the report into event_registration_report.txt. 
9.End the program
